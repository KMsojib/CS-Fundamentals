# 🔁 Conditional Statements

Conditional statements control the flow of execution based on conditions.

## 🧠 Types
- `if`
- `else if`
- `else`

## 🧪 Sample Code
```c
int marks = 85;
if (marks >= 80)
    printf("Grade: A");
else if (marks >= 60)
    printf("Grade: B");
else
    printf("Grade: C");
